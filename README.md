# EdgeNotes

EdgeNotes is a Windows top-edge Markdown notes drawer built with React, Tauri, and Tiptap.

## Download

Windows installers are published on the [Releases](https://github.com/15729580683cp-rgb/edge-notes/releases) page.

## Development

```bash
npm install
npm run dev
```

To build the web frontend:

```bash
npm run build
```
